## コーディングルール

インデントはスペース4つ
SQLの予約後は大文字

## 技術構成

- Frontend: React / Next.js
- Backend: Supabase
- Database: MySQL
- Map: Google Maps API or Mapbox
- Auth: Supabase Auth
- Deploy: Vercel

## Git

- コミットメッセージ: Conventional Commits 準拠
  - `feat:`, `fix:`, `docs:`, `refactor:`, `test:`, `chore:`
- ブランチ名: `feature/xxx`, `fix/xxx`
- PRは必ずテストを通してから作成
